SetLink

SetLink(int link [, float y [, int page]])

Deskripsi

Menentukan halaman dan posisi sebuah link.

Parameter

link
Nilai identitas link yang dikembalikan oleh AddLink().
y
Nilai ordinat posisi target; -1 menunjukkan posisi sekarang. Nilai default adalah 0 (bagian atas halaman).
page
Nomor halaman target; -1 menunjukan halaman sekarang. Ini adalah nilai default.

Lihat juga

AddLink().
Indeks
