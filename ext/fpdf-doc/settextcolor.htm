SetTextColor

SetTextColor(int r [, int g, int b])

Deskripsi

Menentukan warna teks. Nilainya dapat diberikan dalam komponen RGB atau gray scale. Metode ini bisa dipanggil sebelum membuat halaman pertama dan nilainya akan dipertahankan dari halaman ke halaman.

Parameter

r
Jika g dan b diberikan, maka dianggap sebagai komponen warna merah; jika tidak, menandakan derajat abu-abu (grey level). Nilainya antara 0 dan 255.
g
Komponen warna hijau (antara 0 dan 255).
b
Komponen warna biru (antara 0 dan 255).
